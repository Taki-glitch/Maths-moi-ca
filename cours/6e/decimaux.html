🔢 Nombres décimaux

Niveau : 6ème

📘 Cours

Un nombre décimal s’écrit avec une virgule. Il peut aussi s’écrire sous forme de fraction décimale (dénominateur 10, 100, 1000...).
Valeur des chiffres : chaque chiffre dépend de sa position (unités, dixièmes, centièmes, millièmes...).
Exemple : dans 47,305 le 3 vaut trois dixièmes.
Pour comparer deux décimaux, on aligne les virgules puis on compare de gauche à droite.
5,2 = 5,20 ; et 5,20 > 5,03.

✏️ Exercices d'entraînement

  1. Donne la partie entière et la partie décimale : 18,73 ; 0,405 ; 92,0.
  2. Écris en lettres : 3,4 puis 12,08.
  3. Complète : 6,7 = 6 + .../10 ; 9,34 = 9 + .../100.
  4. Range dans l’ordre croissant : 4,09 ; 4,9 ; 4,12 ; 4,101.
  5. Compare avec < ou > : 7,05 ... 7,5 ; 10,03 ... 10,030 ; 2,99 ... 3,01.
  6. Calcule : 12,5 + 3,75 ; puis 20 - 4,6.
  1. 18,73 : entière 18, décimale 73 ; 0,405 : entière 0, décimale 405 ; 92,0 : entière 92, décimale 0.
  2. 3,4 : trois unités et quatre dixièmes ; 12,08 : douze unités et huit centièmes.
  3. 6,7 = 6 + 7/10 ; 9,34 = 9 + 34/100.
  4. 4,09 ; 4,101 ; 4,12 ; 4,9.
  5. 7,05 < 7,5 ; 10,03 = 10,030 ; 2,99 < 3,01.
  6. 12,5 + 3,75 = 16,25 ; 20 - 4,6 = 15,4.
