📈 Statistiques

Niveau : 6ème

📘 Cours

En statistiques, on collecte des données, on les organise (tableau, diagramme), puis on les interprète.
Vocabulaire :
  • Effectif : nombre d’occurrences d’une valeur ;
  • Total : somme des effectifs ;
  • Fréquence : effectif / total.
La moyenne se calcule par : somme des valeurs / nombre de valeurs.
Notes : 8 ; 12 ; 10 ; 10. Moyenne = (8+12+10+10)/4 = 10.

✏️ Exercices d'entraînement

  1. Dans une classe, couleurs préférées : Rouge 6, Bleu 10, Vert 4. Quel est l’effectif total ?
  2. Calcule la fréquence du bleu dans l’exercice 1.
  3. Notes obtenues : 9 ; 11 ; 15 ; 5 ; 10. Calcule la moyenne.
  4. Tailles (en cm) : 140 ; 142 ; 142 ; 145 ; 148 ; 142. Donne l’effectif de 142.
  5. Vrai/Faux : « La fréquence est toujours entre 0 et 1 » ; « L’effectif peut être négatif ».
  6. Une série contient 12 valeurs. Une modalité a une fréquence 1/4. Quel est son effectif ?
  1. Total = 6 + 10 + 4 = 20.
  2. Fréquence du bleu = 10/20 = 0,5 (soit 50 %).
  3. Moyenne = (9 + 11 + 15 + 5 + 10) / 5 = 50/5 = 10.
  4. L’effectif de 142 est 3.
  5. Vrai ; Faux.
  6. Effectif = 12 × 1/4 = 3.
