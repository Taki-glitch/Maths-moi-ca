➗ Fractions

Niveau : 6ème

📘 Cours

Une fraction représente une ou plusieurs parts d’un tout partagé en parts égales. Le nombre du haut est le numérateur ; celui du bas, le dénominateur.
  • Si le numérateur est plus petit que le dénominateur, la fraction est inférieure à 1.
  • Si le numérateur est égal au dénominateur, la fraction vaut 1.
  • Si le numérateur est plus grand, la fraction est supérieure à 1.
Fractions décimales : dénominateur 10, 100, 1000... Elles permettent de passer aux nombres décimaux.
3/4 signifie « 3 parts sur 4 ». 7/10 = 0,7.

✏️ Exercices d'entraînement

  1. Écris la fraction correspondant à : 5 parts sur 8 ; 11 parts sur 12.
  2. Dans 9/13, donne le numérateur et le dénominateur.
  3. Complète avec <, > ou = : 3/7 ... 1 ; 10/10 ... 1 ; 14/9 ... 1.
  4. Associe : 2/10 ; 45/100 ; 7/1000 avec « deux dixièmes », « quarante-cinq centièmes », « sept millièmes ».
  5. Écris sous forme décimale : 6/10 ; 3/100 ; 125/100.
  6. Un ruban de 1 m est coupé en 8 parts égales. Zoé prend 3 parts. Quelle fraction du mètre a-t-elle prise ?
  1. 5/8 et 11/12.
  2. Numérateur : 9 ; dénominateur : 13.
  3. 3/7 < 1 ; 10/10 = 1 ; 14/9 > 1.
  4. 2/10 → deux dixièmes ; 45/100 → quarante-cinq centièmes ; 7/1000 → sept millièmes.
  5. 6/10 = 0,6 ; 3/100 = 0,03 ; 125/100 = 1,25.
  6. Elle prend 3/8 du mètre.
