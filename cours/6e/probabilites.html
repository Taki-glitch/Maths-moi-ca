🎲 Probabilités

Niveau : 6ème

📘 Cours

Une probabilité mesure la chance qu’un événement se produise. Elle est comprise entre 0 et 1.
  • 0 : événement impossible ;
  • 1 : événement certain ;
  • entre 0 et 1 : événement possible.
Dans une situation d’équiprobabilité : probabilité = (nombre de cas favorables) / (nombre de cas possibles).
Dé équilibré : P(obtenir 6) = 1/6. P(obtenir un nombre pair) = 3/6 = 1/2.

✏️ Exercices d'entraînement

  1. Dans un lancer de dé, quelle est la probabilité d’obtenir 3 ?
  2. Dans un lancer de dé, quelle est la probabilité d’obtenir un nombre supérieur à 4 ?
  3. Une roue contient 8 secteurs égaux : 3 rouges, 5 bleus. Donne P(rouge) et P(bleu).
  4. Un sac contient 2 billes vertes, 4 jaunes, 4 noires. Probabilité de tirer une bille jaune ?
  5. Classe ces événements : impossible / certain / possible :
    • Tirer 7 avec un dé à 6 faces.
    • Obtenir une lettre de l’alphabet en tirant au hasard parmi A, B, C.
    • Obtenir pile ou face en lançant une pièce.
  6. Transforme en fraction simplifiée : 4/12 ; 6/10.
  1. 1/6.
  2. Nombres > 4 : 5 ou 6, donc 2/6 = 1/3.
  3. P(rouge)=3/8 ; P(bleu)=5/8.
  4. Total 10 billes, jaunes 4 : 4/10 = 2/5.
  5. Impossible ; certain ; possible (et même 1).
  6. 4/12 = 1/3 ; 6/10 = 3/5.
