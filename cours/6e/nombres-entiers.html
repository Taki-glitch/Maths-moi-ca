🔢 Nombres entiers

Niveau : 6ème

📘 Cours

Un nombre entier est un nombre sans partie décimale : ...,-3,-2,-1,0,1,2,3,...
Comparer des entiers : sur une droite graduée, le nombre le plus à droite est le plus grand.
Ranger : ordre croissant (du plus petit au plus grand) ou décroissant.
Repères utiles :
  • 0 est l’origine ;
  • les nombres positifs sont à droite de 0 ;
  • les nombres négatifs sont à gauche de 0.
Exemple : ranger -7 ; 4 ; 0 ; -2 ; 9 dans l’ordre croissant : -7 ; -2 ; 0 ; 4 ; 9.

✏️ Exercices d'entraînement

  1. Recopie les nombres entiers : 4 ; 2,5 ; -8 ; 0 ; 1/2 ; 13.
  2. Range dans l’ordre croissant : 12 ; -3 ; 0 ; -15 ; 7.
  3. Range dans l’ordre décroissant : -4 ; 6 ; 1 ; -9 ; 10.
  4. Complète avec < , > ou = :
    • -5 ... -2
    • 0 ... -1
    • 8 ... 8
  5. Sur une droite graduée, place : -3 ; 2 ; 5 ; -1.
  6. Problème : en montagne, un refuge est à +1850 m et un lac à +1240 m. Quelle est la différence d’altitude ?
  1. Entiers : 4 ; -8 ; 0 ; 13.
  2. -15 ; -3 ; 0 ; 7 ; 12.
  3. 10 ; 6 ; 1 ; -4 ; -9.
  4. -5 < -2 ; 0 > -1 ; 8 = 8.
  5. De gauche à droite : -3 ; -1 ; 2 ; 5.
  6. 1850 - 1240 = 610 m.
