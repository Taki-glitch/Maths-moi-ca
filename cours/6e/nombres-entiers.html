🔢 Nombres entiers

Niveau : 6ème

Accueil → 6ème → Nombres entiers
Les nombres entiers sont les nombres sans virgule : positifs, négatifs ou nuls.
Les nombres entiers peuvent être comparés, additionnés, soustraits et multipliés.
−3 ; 0 ; 7 sont des nombres entiers.

✏️ Exercices

  1. Donner deux nombres entiers positifs.
  2. Classer −2, 5 et 0 dans l’ordre croissant.
  1. Exemples : 3 et 8
  2. −2 ; 0 ; 5
